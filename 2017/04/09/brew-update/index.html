发表于 2017-04-09 | 分类于 tools | 阅读次数

brew更新软件时常用的操作

安装 brew

ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/Go/install)"

搜索

brew search MySQL

查看程序具体信息

brew info mysql

更新 brew

brew update

更新软件

brew upgrade mysql
如果不指定更新哪个则会更新所有可以更新的软件

清理缓存(更新完后的操作)

brew cleanup

查看本地所有可更新的工具

brew outdated
