之前都是用的image sprite或者使用的 Font Awesome这类的开源库
现在咱们已经走进了现代浏览器的年代(ie9+),可以玩点新鲜的东西了,svg

svg 有几个好处:

  • 矢量图,不失真
  • 可以受CSS样式影响,就像一个文本
  • http请求低了
